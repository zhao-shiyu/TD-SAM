# TD-SAM: Temporal and Distance-Guided Adaptations of SAM for Accurate Surgical Instrument Segmentation

This is a PyTorch implementation of the paper [TD-SAM: Temporal and Distance-Guided Adaptations of SAM for Accurate Surgical Instrument Segmentation].

## Usage
#### Environmental Requirements
- Ubuntu 20.04
- Anaconda
- Python=3.10.12
- torch==2.0.1
- cuda==11.7

#### Installation
Clone this repository and then install the dependencies.
```sh
git clone https://github.com/zhao-shiyu/TD-SAM.git
conda create -n tdsam python=3.10.12
conda activate tdsam
conda install pytorch torchvision torchaudio pytorch-cuda=11.7 -c pytorch -c nvidia
cd TD-SAM
pip install -r requirements.txt
```
